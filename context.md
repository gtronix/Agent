# Context notes (optional)

Use this file for **human or agent scratch notes** that are not part of the formal handoff:

- Short-term goals for the next session
- Credentials or URLs placeholders (never store real secrets here)
- Links to tickets or designs

The canonical handoff remains **`context/session_history.md`** and **`context/feature_list.json`**. Prefer updating those for anything the next agent must not miss.
